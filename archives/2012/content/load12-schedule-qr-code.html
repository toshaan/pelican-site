LOAD (Linux Open Administration Days) is an event geared towards Linux sysadmins and people interested in linux administration in general.
With this event we want to provide a platform for linux sysadmins to get together and share their knowledge.

This is a free event, and therefore there's no entrance fee

LOAD12 schedule QR code

One of our speakers asked us if we could create a QR code that links to the url containing the schedule's xml.

So we did :)

You can use this url with Giggity which is an Android app you can use to load the schedule on your phone.
