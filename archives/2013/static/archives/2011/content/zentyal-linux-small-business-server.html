Zentyal, Linux small business server

Submitted by admin on Wed, 2011-03-16 18:16
Presenter(s): 
Jorge Salamero Sanz

Zentyal (formerly eBox Platform) is a Linux small business server, that can
act as Gateway, Infrastructure Manager, Unified Threat Manager, Office Server,
Unified Communication Server or a combination of them. Zentyal Server is an
open source project, released under the GNU General Public License (GPL) and
it runs on top of Ubuntu GNU/Linux.

With Zentyal you are able to manage network services like advanced routing,
firewall and traffic shaping, DHCP, DNS, RADIUS, VPN, HTTP proxy, IDS, mail,
groupware, file and printer sharing, VoIP, IM and more. It offers a web
interface to configure well-known and popular software like the Linux
networking stack (iptables, tc, ip), ISC DHCP, Bind9, FreeRADIUS, OpenVPN
and OpenSwan, Squid, Snort, Postfix, Dovecot and Zarafa, Apache, OpenLDAP,
Samba and CUPS, Asterisk or eJabberd.

All these services are not only configured via a unified interface, but Zentyal
framework makes them to work together integrating the functionalities between
different services, like for example modules ask the firewall to open required
ports or to the DNS to create the needed records. The CA provides them SSL
certificates, Apache configures the specified virtualhost where to deploy the
web application and all user information is stored on OpenLDAP, etc.

This technology allows busy or novice system administrators to quickly deploy
complex, but well tested configurations from a unified interface, automating
mosts tasks, helping to avoid mistakes and saving system administrators' time.
Also the required time to maintain a server is cut down because of the
centralized logs, events, monitoring and alerts management.

Furthermore, Zentyal is very flexible tool that allows you to customize
configuration templates, being non intrusive if you want to integrate it with
your own solution and in addition, it is easy to extend to manage new services
using its framework designed for service configuration and management.

This 45-minute presentation will consist on presentation of the project, with
some practical examples on how to configure some specific network services. In
addition, the Zentyal development framework is presented as well as the
challenges the Zentyal Development Team faces developing the tool and what
quality assurance policies the development follows.
